# Ajinkya Patil — Portfolio v3.0

## 📁 Folder Structure
```
ajinkya-portfolio/
├── index.html              ← Main HTML (no inline styles)
├── css/
│   └── style.css           ← All styles
├── js/
│   └── script.js           ← All JavaScript
└── assets/
    ├── images/
    │   ├── Ajinkya.png     ← ✅ YOUR PHOTO (place here!)
    │   ├── project1.png    ← QC Management screenshot
    │   ├── project2.png    ← Electronic Shop screenshot
    │   └── project3.png    ← 3rd project screenshot
    ├── videos/
    └── docs/
        └── Ajinkya_Patil_Resume.pdf
```

## ✅ How to Add Your Photo
Your photo path is already set in index.html:
  <img src="assets/images/Ajinkya.png" ...>

Just copy your photo into: assets/images/Ajinkya.png
(Any format works: .png, .jpg, .jpeg — just rename it to Ajinkya.png)

## ✅ How to Add Project Screenshots
Replace the placeholder block inside each .project-card__image div:
  <img src="assets/images/project1.png" alt="QC Management System">

## 🌐 Deploy Free
- Netlify: drag & drop folder at netlify.com (30 seconds)
- GitHub Pages: push to repo named ajinkyapatil55.github.io
